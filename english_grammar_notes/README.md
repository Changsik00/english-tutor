# 영문법 기초 완성 노트

문법 규칙을 원리로 이해하고, 실생활에서 바로 쓸 수 있도록 구성한 자체 제작 영어 학습 자료입니다.
성인 재학습자(학교에서 배운 문법을 다시 기초부터 다지고 싶은 분)를 기본 대상으로 삼았지만, 표준 문법 전 범위를 다루므로 누구나 참고할 수 있습니다.
(참고: 문법 챕터의 유닛 구성은 YouTube 채널 grammarTV의 "대체불가 중학 영문법" 재생목록의 목차 순서를 참고했으며,
설명·예문·연습문제·부가 자료는 모두 새로 작성한 오리지널 콘텐츠입니다.)

## 전체 구성

```
english_grammar_notes/
├── README.md          (이 파일)
├── 용어사전.md          문법 용어 70여 개 정리
└── chapters/           문법 챕터 16개(ch01~16) + 실용 챕터 2개(ch17~18)
```

각 문법 챕터(ch01~16) 안에는 다음이 모두 통합되어 있습니다 (더 이상 원리·진단평가·모의고사가 별도 파일로 분리되어 있지 않습니다):

- **학습목표** — 이 챕터를 마치면 할 수 있어야 하는 것
- **이 챕터와 관련된 원리** — 그 챕터의 문법이 왜 그런 식으로 작동하는지, 영어를 관통하는 핵심 원리(어순, 정보흐름, 동사중심성, 압축과 확장, 심리적 거리)로 설명
- **유닛별 학습** — 핵심 설명 → 예문(다수) → 주의사항 → 연습문제 → 정답
- **원리 적용 연습** — 시험 문제 형식이 아니라, 문제 바로 뒤에 "왜 그런지"를 원리와 연결해 설명하는 이해 확인 연습
- **챕터 핵심 정리** — 규칙을 표/불릿으로 압축 정리
- (파트 마지막 챕터에는) **여기까지 왔다면 / 교재를 마쳤다면** — 복습 주기와 오답노트 활용 팁

## 학습 순서 (권장)

1. ch01부터 순서대로 진행 — 각 챕터 서두의 "이 챕터와 관련된 원리"를 먼저 읽으면 그 챕터의 규칙이 왜 그런지 이해하고 들어갈 수 있습니다.
2. 유닛별 설명·예문을 읽고 연습문제를 먼저 풀어본 뒤 정답을 확인합니다.
3. 챕터 끝의 "원리 적용 연습"과 "챕터 핵심 정리"로 마무리합니다.
4. ch04, ch09, ch13, ch16 끝에 있는 복습 팁에 따라 3일/7일 뒤 해당 구간을 다시 훑어봅니다.
5. 문법 16챕터를 마친 뒤 **ch17(실생활 표현)**, **ch18(실용 글쓰기)** 로 실전 활용력을 보강합니다.
6. 헷갈리는 용어는 **[용어사전.md](용어사전.md)** 에서 바로 확인합니다.
