# 학습 가이드

이 문서는 『중학 영문법 학습 노트』(16개 챕터, 107개 유닛)를 어떻게 공부하면 효과적인지 안내합니다. 교재를 처음 펼치기 전에 한 번 읽고, 학습 중간중간 다시 참고하세요.

---

## 1. 교재 구성 한눈에 보기

이 교재는 **진단평가 → 16개 챕터(4개 파트) → 파트별 총정리 모의고사**의 흐름으로 구성되어 있습니다.

1. **진단평가**: 본격적인 학습 전, 현재 실력을 점검하고 취약한 영역을 파악합니다.
2. **16개 챕터 학습**: 아래처럼 4개 파트로 묶여 있으며, 각 파트는 문법적으로 연결된 주제끼리 모아둔 것입니다.
3. **파트별 총정리 모의고사**: 한 파트를 마칠 때마다 배운 내용을 종합적으로 점검하는 모의고사를 풉니다.

| 파트 | 챕터 | 주제 | 핵심 내용 |
|---|---|---|---|
| **Part 1** | ch01~04 | 기초 문법 | 문장 구조, 8품사, 문장 성분, 문장의 형식, 시제 |
| **Part 2** | ch05~09 | 태와 준동사 | 수동태, 조동사, to부정사, 동명사, 분사 |
| **Part 3** | ch10~13 | 품사 심화와 연결어 | 관사·대명사·형용사·부사, 비교, 전치사, 접속사 |
| **Part 4** | ch14~16 | 관계사와 특수구문 | 관계사, 가정법, 특수구문 |

**왜 이 순서인가?** Part 1에서 문장을 읽고 만드는 기본기를 잡고, Part 2에서 문장을 더 풍부하게 확장하는 도구(태·준동사)를 배우고, Part 3에서 문장을 꾸미고 연결하는 요소를, Part 4에서 문장을 두 개 이상 엮거나 특수한 형태로 바꾸는 방법을 배웁니다. 즉 뒤로 갈수록 앞 파트의 내용을 전제로 하므로, **순서를 건너뛰지 말고 차례대로 학습하는 것을 권장합니다.**

## 2. 12주 학습 진도표

아래는 하루 30~50분, 주 5~6일 학습을 기준으로 한 표준 진도표입니다. 총 12주 중 **7주는 새 내용 학습, 4주는 파트별 모의고사+복습, 1주는 전체 총복습**으로 구성했습니다. 개인 진도에 따라 ±1~2주 조정해도 좋습니다.

| 주차 | 학습 범위 | 비고 |
|---|---|---|
| 1주 | ch01 기본구조와품사 (Unit 001~009) + ch02 문장의종류 (Unit 010~014) | Part 1 시작 |
| 2주 | ch03 문장의형식 (Unit 015~019) + ch04 시제 (Unit 020~027) | Part 1 마무리 |
| 3주 | **Part 1 총정리 모의고사** + 오답노트 정리 + 취약 유닛 재학습 | 복습 주간 |
| 4주 | ch05 수동태 (Unit 028~033) + ch06 조동사 (Unit 034~038) | Part 2 시작 |
| 5주 | ch07 to부정사 (Unit 039~047) + ch08 동명사 (Unit 048~051) + ch09 분사 (Unit 052~058) | 준동사 3형제, 분량이 많은 주 — 필요시 이틀 더 배분 |
| 6주 | **Part 2 총정리 모의고사** + 오답노트 정리 + 취약 유닛 재학습 | 복습 주간 |
| 7주 | ch10 관사·대명사·형용사·부사 (Unit 059~067) + ch11 비교 (Unit 068~072) | Part 3 시작 |
| 8주 | ch12 전치사 (Unit 073~076) + ch13 접속사 (Unit 077~088) | Part 3 마무리 |
| 9주 | **Part 3 총정리 모의고사** + 오답노트 정리 + 취약 유닛 재학습 | 복습 주간 |
| 10주 | ch14 관계사 (Unit 089~097) + ch15 가정법 (Unit 098~100) + ch16 특수구문 (Unit 101~107) | Part 4 전체, 분량이 많은 주 — 필요시 이틀 더 배분 |
| 11주 | **Part 4 총정리 모의고사** + 오답노트 정리 + 취약 유닛 재학습 | 복습 주간 |
| 12주 | **전체 총복습**: 16개 챕터 핵심 정리 다시 읽기 + 오답노트 전체 재점검 + 헷갈리는 유닛 최종 재학습 | 총정리 주간 |

> 진도가 빠듯하게 느껴지면 5주차와 10주차를 각각 2주로 나눠 총 14주 계획으로 조정해도 무방합니다. 중요한 것은 속도가 아니라 **각 파트를 마칠 때 총정리 모의고사에서 확인 없이 넘어가지 않는 것**입니다.

## 3. 복습 주기 가이드 (에빙하우스 망각곡선 적용)

사람은 새로 배운 내용을 하루 안에 상당 부분 잊어버립니다. 이를 막는 가장 효과적인 방법은 **잊혀지기 직전에 다시 떠올리는 것(능동적 회상, active recall)**입니다. 이 교재에서는 각 유닛을 학습한 뒤 다음 주기로 복습할 것을 권장합니다.

| 복습 시점 | 방법 |
|---|---|
| **학습 당일** | 유닛의 핵심 설명과 예문을 소리 내어 읽고, 연습문제를 푼다. |
| **1일 후** | 예문과 핵심 규칙만 다시 보고, 전날 틀린 문제를 다시 풀어본다. |
| **3일 후** | 책을 보지 않고 핵심 규칙을 떠올려본 뒤(백지 인출), 맞았는지 확인한다. |
| **7일 후** | 해당 챕터의 연습문제를 처음부터 다시 풀어본다. |
| **30일 후** | 오답노트만 훑어보며 여전히 헷갈리는 부분이 있는지 최종 점검한다. |

## 4. 오답노트 작성법

오답노트는 단순히 틀린 문제를 옮겨 적는 것이 아니라, **"왜 틀렸는지"와 "다음에 어떻게 판단할지"를 언어로 정리하는 작업**입니다. 아래 순서를 따르세요.

1. **틀린 문제를 그대로 옮겨 적는다.** 문제 번호와 챕터/유닛을 함께 기록해 나중에 원문을 다시 찾아볼 수 있게 합니다.
2. **틀린 이유를 스스로 분석한다.** 다음 중 어디에 해당하는지 구분해보면 도움이 됩니다.
   - 개념을 몰랐다 (규칙 자체를 학습하지 않음)
   - 개념은 알았지만 문장에 적용하지 못했다 (적용 실수)
   - 비슷한 규칙과 헷갈렸다 (혼동)
   - 문제를 잘못 읽었다 (단순 실수)
3. **핵심 규칙을 나의 언어로 정리한다.** 교재 설명을 그대로 베끼지 말고, 스스로 이해한 방식으로 한두 문장으로 요약합니다.
4. **비슷한 유형의 예문을 하나 더 만들어본다.** 직접 문장을 만들어보면 개념이 진짜 내 것이 되었는지 확인할 수 있습니다.
5. **주기적으로 다시 훑어본다.** 위 복습 주기(1일/3일/7일/30일 후)에 맞춰 오답노트만 따로 다시 읽습니다. 특히 모의고사 직전에는 오답노트 전체를 한 번 훑는 것이 큰 도움이 됩니다.

```
| 날짜 | 챕터/유닛 | 문제 | 틀린 이유 | 정리 |
|---|---|---|---|---|

```

| 날짜 | 챕터/유닛 | 문제 | 틀린 이유 | 정리 |

## 5. 효과적인 학습 팁

1. **예문은 눈으로만 읽지 말고 소리 내어 읽는다.** 발음과 함께 문장 구조가 몸에 익으면 서술형이나 영작 문제에서 훨씬 빠르게 문장을 만들 수 있습니다.
2. **설명을 읽기 전에 연습문제부터 풀어본다.** 틀리더라도 먼저 스스로 답을 예측해보는 과정(능동적 회상)이 그냥 설명을 읽는 것보다 기억에 훨씬 오래 남습니다.
3. **서술형·영작 문제는 반드시 손으로 직접 써본다.** 눈으로 보고 "알 것 같다"고 넘어가는 것과 실제로 써서 맞히는 것은 완전히 다른 능력입니다. 특히 어순이 헷갈리는 준동사·관계사 단원에서는 손으로 쓰는 연습이 필수입니다.
4. **배운 개념을 자기 말로 설명해본다.** 친구나 가족에게 "이 문법이 왜 이렇게 쓰이는지" 설명해보세요. 막히는 부분이 있다면 그 부분이 아직 완전히 이해되지 않은 것입니다.
5. **타이머를 켜고 실전처럼 모의고사를 푼다.** 파트별 총정리 모의고사는 제한 시간을 정해두고 풀어야 실제 시험 감각을 익힐 수 있습니다.
6. **오답은 다음 날 반드시 다시 풀어본다.** 틀린 문제를 그날 바로 다시 풀면 정답을 기억해서 맞히는 경우가 많으므로, 하루 정도 간격을 두고 다시 풀어야 진짜 이해했는지 확인할 수 있습니다.
7. **비슷한 개념끼리 비교표를 직접 만들어본다.** 예를 들어 to부정사 vs 동명사, 관계대명사 vs 관계부사처럼 헷갈리기 쉬운 짝을 표로 정리하면 파트 총정리 모의고사 대비에 큰 도움이 됩니다.

---

이 가이드대로 12주를 완주하면 16개 챕터, 107개 유닛을 체계적으로 익히고 오답노트까지 완성할 수 있습니다. 계획대로 안 되는 주가 있어도 괜찮습니다. 중요한 것은 멈추지 않고 복습 주기를 지키며 끝까지 완주하는 것입니다.
