# 영문법 기초 완성 노트

문법 규칙을 원리로 이해하고, 실생활에서 바로 쓸 수 있도록 구성한 자체 제작 영어 학습 자료입니다.
성인 재학습자(학교에서 배운 문법을 다시 기초부터 다지고 싶은 분)를 기본 대상으로 삼았지만, 표준 문법 전 범위를 다루므로 누구나 참고할 수 있습니다.
(참고: 문법 챕터의 유닛 구성은 YouTube 채널 grammarTV의 "대체불가 중학 영문법" 재생목록의 목차 순서를 참고했으며,
설명·예문·연습문제·모의고사·부가 자료는 모두 새로 작성한 오리지널 콘텐츠입니다.)

## 전체 구성

```
middle_school_grammar/
├── README.md                    (이 파일)
├── 00_영어의원리.md               문법을 관통하는 5가지 핵심 원리 (암기 대신 이해)
├── 00_학습가이드.md               학습 순서, 12주 진도표, 복습 주기, 오답노트 템플릿
├── 00_진단평가.md                 시작 전 실력 점검용 진단평가 40문항 + 추천 학습 전략
├── 용어사전.md                    문법 용어 70여 개 정리
├── chapters/                     문법 챕터 16개(ch01~16) + 실용 챕터 2개(ch17~18)
└── mock_exams/                   파트별(4개) 총정리 모의고사 — 중간/기말고사 스타일
```

## 학습 순서 (권장)

1. **[00_영어의원리.md](00_영어의원리.md)** 로 영어 문법을 관통하는 원리부터 파악 — 이후 챕터를 공부할 때 "왜 이런 규칙인지"가 이해됨
2. **[00_진단평가.md](00_진단평가.md)** 로 현재 실력 점검
3. **[00_학습가이드.md](00_학습가이드.md)** 의 진도표에 따라 챕터 순서대로 학습
4. 각 문법 챕터(ch01~16): 학습목표 확인 → 유닛별 설명·예문 학습 → 연습문제 풀이 → 챕터 핵심 정리로 복습
5. 파트(4챕터 내외)가 끝날 때마다 **mock_exams/** 의 해당 파트 총정리 모의고사로 점검
6. 문법 16챕터를 마친 뒤 **ch17(실생활 표현)**, **ch18(실용 글쓰기)** 로 실전 활용력 보강
7. 헷갈리는 용어는 **[용어사전.md](용어사전.md)** 에서 바로 확인

## 파일 구조 안내

- 각 문법 챕터 파일: `학습목표(체크리스트) → 유닛별 [핵심 설명 → 예문 → 주의사항 → 연습문제 → 정답] → 챕터 핵심 정리`
- 각 모의고사 파일: `객관식 문제 → 서술형 문제 → 정답 및 상세 해설`
- 연습문제는 먼저 풀어본 뒤 정답을 확인하는 방식으로 학습하는 것을 추천합니다.
