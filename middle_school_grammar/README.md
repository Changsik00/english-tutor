# 중학 영문법 학습 노트

중학교 영문법 전 범위를 16개 챕터, 107개 유닛으로 정리한 자체 제작 학습 자료입니다.
(참고: 유닛 구성은 YouTube 채널 grammarTV의 "대체불가 중학 영문법" 재생목록의 목차 순서를 참고했으며,
설명·예문·연습문제는 모두 새로 작성한 오리지널 콘텐츠입니다.)

## 사용법

- 각 챕터 파일은 유닛별로 `핵심 설명 → 예문 → 주의사항 → 연습문제 → 정답`순으로 구성되어 있습니다.
- 연습문제를 먼저 풀어본 뒤 정답을 확인하는 방식으로 학습하는 것을 추천합니다.
